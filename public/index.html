웹 페이지를 구조화 할수 있습니다. 몇 개의 프로젝트를 하며 어떤 구조가 필요한지 자각하고 있으며 웹 표준,접근성에 알맞은 구조를 만들기 위해 노력하고 있습니다.

html

하고자 하는 디자인을 직접 적용할수 있습니다. 좀더 다양하고 자유로운 디자인을 적용하기 위하여 노력중이며, 화려해 보이지만 꼭 필요한 디자인과 안전성 있는 interactive web을 목표로 하고 있습니다.

css

동작을 수행할수 있습니다. 어떤 행동에 대하여 무엇이 필요한지 설계 할수 있으며, 해당 설계를 바탕으로 직접 구현할수 있습니다. es2015+를 바탕으로 진행하고 있으며 함수형 프로그래밍을 통하여 코드의 복잡성과 오류를 줄이는 것을 목표로 하고 있습니다.

js

SPA를 만들수 있습니다. react의 동작원리, life-cycle을 이해하고 있으며 Hook을 기준으로 사용하고 있습니다. 두개의 프로젝트 모두 react를 적용하였으며 그 중 하나는 next.js를 사용하지 않고, 순수하게 react SSR을 구현하였습니다.

react

데이터를 관리 할수 있습니다. react에서 side effect를 관리하기 위하여 redux-saga를 사용하고 있습니다. api 서버로의 요청을 react element에서 분리하여 rest api를 요청하기 때문에 깔끔한 코드를 작성할수 있습니다.

redux

CRUD를 사용할수 있습니다. mysql을 바탕으로 기본적인 crud를 사용할수 있으며, sequelize를 통하여 modeling을 할수 있습니다.

sql

api 서버를 만들수 있습니다. express를 사용하여 기본적인 api 백엔드 서버를 만들어 DB와 연동하여 프론트와 작업을 할수 있습니다.

node
